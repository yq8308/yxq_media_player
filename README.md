# yxq_media_player
    基于ijkplayer实现的一个demo

    播放器和弹幕都是用的bilibili的开源代码
    yxq_media_player ： android java code. （Demo效果的实现代码）
    APK: 生成好的apk demo （仿斗鱼app）
    server: 一个简单的c++聊天通信服务端，主要实现消息的接受转发（没有对消息内容重新封装，只是原封不动的发出去），一对多
